# scilab-control-system-toolbox-functions

This project is a collection of control system functions implemented in Scilab, developed as part of the FOSSEE fellowship for Control System Toolbox development.

The goal is to take functions from octave's `control` package and reimplement them natively in scilab

## CONTENTS

Each function has its own folder containing:
- a `.sci` file with the implementation and test cases
- a `README.md` explaining the function, parameters, and usage

all functions are made with inbuilt scilab functions
test cases run automatically and print results to the console
